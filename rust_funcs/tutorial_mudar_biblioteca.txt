Como alterar esta biblioteca em Rust para Python:

Alterar:
 -Acesse a pasta do arquivo lib.rs, LSTM\rust_funcs\src\lib.rs.
 -Basta alterar o arquivo e atualizar (veja abaixo).
 -Lembre-se de adicionar funções a fn rust_funcs (por exemplo, m.add_function(wrap_pyfunction!(logloss, m)?)?;).

Atualizar:
 -Acesse a pasta deste arquivo (atualmente, LSTM\rust_funcs).
 -Abra um cmd nesta pasta.
 -Execute '.env\Scripts\activate' (se já estiver na pasta rust_funcs).
 -----Certifique-se de que não há um arquivo python utilizando a biblioteca (reiniciar o kernel é suficiente no caso de notebooks).
 -Execute 'maturin develop' ou, preferencialmente, 'maturin develop --release' para ativar otimizações.
 -A biblioteca foi atualizada :)
